# ITAI ML First Repository

This repository was created for the **Classic Machine Learning** course as part of the Getting Started with GitHub & Your First Repository assignment.

## Assignment Purpose

The purpose of this assignment is to practice using GitHub as a platform for storing, sharing, and submitting machine learning coursework. GitHub is commonly used by data scientists, AI engineers, and developers to organize projects, track changes, and collaborate with others.

## Repository Contents

This repository contains my first machine learning course file:

* `hello_ml.ipynb` — A Google Colab / Jupyter Notebook file that prints a simple machine learning greeting.

## Notebook Description

The notebook includes a basic Python code cell:

```python
print("Hello, Machine Learning!")
```

This simple program was used to practice creating a notebook, running Python code, saving the notebook, and uploading it to GitHub.

## What I Learned

Through this assignment, I learned how to create a GitHub repository, make it public, add a README file, create a notebook in Google Colab, download the notebook as an `.ipynb` file, upload it to GitHub, and submit the repository link through Canvas.

This assignment helped me understand how GitHub will be used as a foundation for submitting and sharing future machine learning projects.
